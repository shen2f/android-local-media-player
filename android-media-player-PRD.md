# 安卓媒体播放器 PRD v1.2

## 1. 文档概述

### 1.1 项目背景
随着移动端视频和音频内容消费增长，用户需要一款轻量、简洁、功能完整的本地媒体播放器，满足日常播放本地音视频的需求，可选择开启在线扩展功能，同时提供流畅的播放体验。

### 1.2 产品目标
- 提供轻量、流畅的本地音视频播放体验
- 支持主流音视频格式
- 简洁直观的交互设计，学习成本低
- 可定制化播放参数满足不同用户需求
- 可选开启在线功能，满足扩展需求，默认关闭保证隐私安全

### 1.3 项目范围
- 核心支持本地媒体文件播放，可选项开启在线流媒体功能
- 支持手机本地存储中的音视频扫描和播放
- 基础播放控制+高级功能选项，完善文件管理和用户自定义能力

---

## 2. 产品定位

**「轻量可定制本地媒体播放器」**
- 目标用户：安卓手机用户，经常观看本地视频/听本地音乐，追求简洁无广告体验，可根据需求开启扩展功能
- 核心价值：快速启动、流畅播放、格式全支持、无广告无推送、隐私可控、功能可扩展

## 3. 功能需求

### 3.0 初始引导逻辑

#### 3.0.1 首次启动引导
- **功能描述**：首次启动App时的权限申请和初始化引导
- **需求细节**：
  - 第一步：欢迎页，介绍产品核心特点「轻量·无广告·隐私安全」
  - 第二步：权限申请，请求存储访问权限，说明用途仅用于读取本地媒体文件
  - 第三步：功能配置选择：
    - 是否允许开启在线功能（默认关闭，用户可手动开启）
    - 是否需要快速扫描常用媒体文件夹（DCIM、Movies、Music等默认目录）
  - 第四步：完成引导，进入主界面
  - 支持跳过引导，直接进入主界面手动配置

#### 3.0.2 权限适配说明
- **Android 8.0 - Android 12**：申请存储读写权限
- **Android 13+**：申请READ_MEDIA_VIDEO / READ_MEDIA_AUDIO / READ_MEDIA_IMAGES细分权限
- 权限说明清晰，告知用户每个权限的用途，不申请非必要权限
- 权限被拒绝时，友好提示用户，引导用户手动选择文件夹

### 3.1 核心功能

#### 3.1.1 媒体库
- **功能描述**：按分类和文件系统索引展示媒体文件
- **需求细节**：
  - 底部四个主Tab：视频、音频、图片、文件夹
  - 默认不做自动全量扫描，提供"全量扫描"按钮，由用户允许后再启动扫描
  - 支持添加自定义扫描目录
  - **文件夹浏览Tab新增**：
    - 按原生文件目录结构浏览
    - 支持显示文件夹内媒体文件数量缩略
    - 支持进入文件夹查看所有媒体文件
    - 支持文件夹级别的收藏/隐藏操作
  - 视频按文件夹/时间/大小排序展示
  - 音频按歌手/专辑/流派分类展示
  - 图片按时间/文件夹排序展示，网格缩略图预览
  - 支持搜索功能，按文件名搜索
  - 支持收藏常用文件/文件夹

#### 3.1.2 视频播放
- **功能描述**：核心视频播放功能
- **需求细节**：
  - 支持格式：MP4、MKV、AVI、RMVB、FLV、WMV、WebM等主流格式
  - 支持硬解码/软解码切换
  - 基础播放控制：播放/暂停、进度拖动、快进快退（双击侧边）
  - 音量/亮度调节（左侧滑动亮度，右侧滑动音量）
  - 缩放模式：自适应、铺满、16:9、4:3、原始大小
  - 旋转屏幕
  - 支持倍速播放：0.25x ~ 4.0x
  - **新增：画中画模式**：
    - 支持安卓原生画中画功能，退到后台可以小窗播放
    - 画中窗支持基础播放控制，拖动调整位置
    - Android 8.0+ 支持

#### 3.1.3 音频播放
- **功能描述**：后台音频播放
- **需求细节**：
  - 支持格式：MP3、AAC、FLAC、WAV、OGG、M4A
  - 后台播放，支持通知栏控制
  - 支持锁屏封面显示
  - 播放列表顺序播放/循环播放/随机播放
  - 支持歌词显示（支持本地lrc文件自动匹配）
  - **新增：自定义歌单**：
    - 用户可创建自定义歌单
    - 支持添加/移除歌曲
    - 支持歌单重命名、删除
    - 支持收藏歌单

#### 3.1.4 播放历史
- **功能描述**：记录播放进度和历史
- **需求细节**：
  - 自动记录每个视频的播放进度，下次打开自动续播
  - 播放历史列表，可快速打开最近播放
  - 支持单条删除/清空全部历史记录

#### 3.1.5 图片浏览
- **功能描述**：本地图片浏览功能
- **需求细节**：
  - 支持格式：JPG、PNG、GIF、WEBP、BMP
  - 网格缩略图展示，支持缩放网格大小
  - 点击图片进入全屏浏览，左右滑动切换图片
  - 支持双指缩放、旋转图片
  - 支持图片分享、设为壁纸
  - 支持GIF动图播放控制

#### 3.1.6 文件管理
- **功能描述**：基础文件管理功能
- **需求细节**：
  - 支持删除不需要的媒体文件
  - 支持重命名文件
  - 支持移动文件到其他文件夹
  - 支持分享文件到其他App
  - 支持隐藏私密文件夹，隐藏后不在媒体库显示
  - 可在设置中查看已隐藏文件夹，支持取消隐藏

### 3.2 扩展功能

- 字幕支持：加载本地srt/ass字幕，支持字幕大小、颜色调整
- 视频截图：播放中一键截图保存
- 画面比例调整：支持自定义裁剪
- 睡眠定时：播放定时关闭
- 深色模式：支持系统深色模式切换，支持手动设置浅色/深色/跟随系统
- 密码保护：可对私密文件夹设置密码/生物识别解锁
- **新增：在线功能可选开关**：
  - 默认关闭，不启用任何在线功能
  - 用户可在设置中手动开启
  - 开启后支持：在线字幕搜索、网络流媒体播放、歌单云同步（可选）
  - 所有在线功能均需用户主动触发，不会后台联网
  - 关闭后完全断网，保证隐私安全

- **新增：画中画播放**：已在视频播放中描述

- **新增：自定义歌单**：已在音频播放中描述

## 4. 交互设计说明

### 4.1 首页交互
- 底部导航栏四个Tab：「视频」「音频」「图片」「文件夹」，点击切换
- 默认进入「视频」 tab，展示所有视频文件，网格布局展示缩略图
- 切换「音频」 tab 展示音频列表，切换「图片」 tab 展示图片网格，切换「文件夹」tab展示文件目录结构
- 右下角浮动按钮，支持手动刷新媒体库/打开文件
- 顶部搜索框点击进入搜索模式，支持输入文件名搜索

### 4.2 视频播放交互
1. **启动**：点击视频文件直接进入全屏播放
2. **控制调出**：单击屏幕中间调出控制栏，再次点击隐藏
3. **手势操作**：
   - 左侧上下滑动 → 调整亮度
   - 右侧上下滑动 → 调整音量
   - 左右横向滑动 → 进度拖动
   - 双击左侧/右侧 → 后退/前进10秒
   - 双指缩放 → 调整画面大小
4. **更多选项**：右上角「...」弹出菜单，包含：倍速、缩放、字幕、解码选择、截图、画中画等功能
5. **画中画触发**：点击Home键自动进入画中画（系统授权后），也可从更多菜单手动触发

### 4.3 音频播放交互
- 点击音频文件进入播放详情页
- 展示专辑封面（支持从文件提取）
- 上一曲/播放暂停/下一曲控制
- 横条拖动进度
- 下拉通知栏保留控制控件
- 右上角菜单可添加到自定义歌单

### 4.4 图片浏览交互
- 点击图片tab进入网格缩略图浏览
- 点击任意图片进入全屏浏览模式
- 左右滑动切换上一张/下一张图片
- 双指缩放放大缩小图片，双击放大到原尺寸
- 长按弹出操作菜单：分享、设为壁纸、删除等

### 4.5 文件夹浏览交互
- 按目录层级展示，顶部显示当前路径
- 支持点击进入子文件夹，点击返回按钮回到上级
- 文件夹显示包含媒体数量
- 长按文件夹弹出操作菜单：添加到媒体库、收藏、隐藏、属性

### 4.6 自定义歌单交互
- 音频Tab顶部增加「歌单」分类选项卡
- 点击进入歌单列表，展示用户创建的所有歌单
- 点击歌单进入歌单详情，展示歌单内所有歌曲
- 支持长按歌曲拖拽调整播放顺序

### 4.7 设置页交互
- 采用安卓标准设置分组布局
- 分组：播放设置、界面设置、文件管理、隐私设置、在线功能、关于
- **在线功能分组**：单独列出，开关默认关闭，清晰说明开启后的联网行为
- 二级页面切换，返回按钮回到上级

## 5. 非功能需求

### 5.1 性能需求
- App安装包体积 < 25MB（包含FFmpeg核心库）
- 冷启动时间 < 1.5秒
- 播放1080p视频CPU占用 < 15%
- 播放4K视频CPU占用 < 30%
- 后台播放音频内存占用 < 100MB
- 媒体库扫描1000个媒体文件时间 < 5秒

### 5.2 兼容性需求
- 支持 Android 8.0 及以上版本
- 适配主流手机屏幕尺寸：手机、折叠屏、平板
- 支持横竖屏切换
- 适配深色/浅色模式切换，无样式bug
- 适配Android 13+ 细分权限模型

### 5.3 隐私需求
- 默认不联网，关闭在线功能时不请求任何网络权限
- 不会上传用户任何媒体数据和个人信息
- 只在开启在线功能时才申请网络权限
- 只需要必要的存储/媒体读取权限，不需要电话、位置等其他敏感权限
- 所有用户数据存储在本地，不云端同步（除非用户主动开启在线云同步功能）

### 5.4 稳定性需求
- 连续播放4小时视频不出现卡顿或闪退
- 播放过程中接听电话后恢复播放正常
- 低内存情况下系统回收后，恢复App能继续播放

## 6. 权限适配清单

| Android版本 | 所需权限 | 用途说明 |
|------------|---------|---------|
| Android 8.0 - 12 | READ_EXTERNAL_STORAGE, WRITE_EXTERNAL_STORAGE | 读取和存储媒体文件 |
| Android 13+ | READ_MEDIA_VIDEO | 读取视频文件 |
| Android 13+ | READ_MEDIA_AUDIO | 读取音频文件 |
| Android 13+ | READ_MEDIA_IMAGES | 读取图片文件 |
| 全部版本（开启在线功能后） | INTERNET | 在线功能网络访问 |
| Android 8.0+ | SYSTEM_ALERT_WINDOW | 画中画模式 |

## 7. 验收标准

1. 核心功能：所有上述核心功能可正常使用，无crash
2. 主流格式播放兼容测试：10种常见格式均可正常播放
3. 性能指标满足上述5.1要求
4. 交互流畅，无明显卡顿，手势操作响应及时
5. 权限适配正确，不同Android版本权限申请流程符合要求
6. 关闭在线功能时，App无任何网络请求
